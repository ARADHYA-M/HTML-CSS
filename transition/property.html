Faculty Development Programme (FDP) Management System

A comprehensive web-based system developed to manage and track faculty development activities such as FDPs, conferences, and guest lectures. The platform includes user authentication, role-based access, and automated text extraction from uploaded certificates. It streamlines data entry, organizes records efficiently in categorized tables, and allows easy export of reports, making it a complete digital solution for academic institutions.

Personal Portfolio Website

A fully responsive personal website built to showcase my skills, projects, and achievements in an elegant and minimal design. It highlights sections like About Me, Projects, and Contact, using modern HTML, CSS, and JavaScript. The website adapts seamlessly to any screen size and serves as both a professional portfolio and a creative expression of my design approach.

Recipe Card Webpage

This project presents cooking recipes in a stylish, card-based design where each card features an image, title, ingredients, and preparation steps. Built with a focus on visual clarity and spacing, the layout uses CSS flexbox to align elements neatly across various screen sizes. It demonstrates attention to detail in typography, margins, and readability, creating a pleasant browsing experience.
